# 🚗 CarRental — сайт аренды автомобилей

**CarRental** — это полнофункциональный веб-сайт для аренды автомобилей, разработанный на ASP.NET Core MVC с использованием Entity Framework Core, SQLite, Identity для аутентификации и Bootstrap 5. Проект выполнен в современном красно-сером стиле с анимациями и адаптивным дизайном.

---

## ✨ Функционал

- **Регистрация и вход** пользователей (ASP.NET Core Identity).
- **Просмотр автомобилей** с фильтрацией по доступности.
- **Бронирование автомобиля** с выбором дат и автоматическим расчётом стоимости.
- **Личный кабинет** — список бронирований текущего пользователя.
- **Отмена бронирования**.
- **REST API** для получения списка автомобилей и создания аренды (документация Swagger).
- **Главная страница** с каруселью всех доступных автомобилей, которая автоматически переключается каждые 5 секунд с плавной анимацией.
- **Страница "О разработчиках"** с контактами: email, GitHub, Telegram.
- **Адаптивный дизайн** для мобильных устройств.

## 🚀 Запуск проекта (локально)

### Требования
- [.NET 8 SDK](https://dotnet.microsoft.com/download) (или новее)
- [Visual Studio 2022](https://visualstudio.microsoft.com/) / VS Code / любой редактор
- [Git](https://git-scm.com/) (опционально)

### Инструкция

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/ваш-username/CarRental.git
   cd CarRental
  
2. **Восстановите зависимости**
    ```bash
    dotnet restore
3. **Создайте базу данных**
   При первом запуске база создастся автоматически благодаря EnsureCreated().
Если нужно пересоздать:
   ```bash
   rm -f carrental.db          # для Linux/macOS
   del carrental.db            # для Windows
4. **Запустите приложение**
   ```bash
   dotnet run
5. **Откройте браузер и перейдите по адресу**
   ```bash
   https://localhost:5001
   ```
   (или http://localhost:5000)

## 🔧 API Endpoints

| Метод | URL                     | Описание                         |
|-------|-------------------------|----------------------------------|
| GET   | `/api/CarsApi`          | Получить список всех автомобилей  |
| GET   | `/api/CarsApi/{id}`     | Получить автомобиль по ID         |
| POST  | `/api/CarsApi/rent`     | Создать бронирование (JSON)       |

Документация API доступна по адресу: `/swagger` (в режиме разработки).
